This program is a short game were blocks falling like a Tetris game.
This code was made on C++ on Visual Studio.

<img width="719" height="688" alt="image" src="https://github.com/user-attachments/assets/ef6bc2f8-b65b-4aa0-84fc-b1ed31644c6a" />


You can move to the left with "Q" and to the right with "D".
You can also jump with "Z".

<img width="787" height="701" alt="image" src="https://github.com/user-attachments/assets/9a2a1708-d968-4103-90c4-a2597be7092a" />


The program start on the menu and you have to click on one the two different button.
An "Start" button who start the game and an "Exit" button who quit the game/the program.

<img width="532" height="427" alt="image" src="https://github.com/user-attachments/assets/a1a15e77-18a6-474b-a325-6dcff5eb2b0d" />

Your goal is to dodge and climb the blocks.

There a scrolling who start to go up when your character are at the middle of the grid.
This scrolling are constantly center on your character.
